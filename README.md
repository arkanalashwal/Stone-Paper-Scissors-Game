# Stone, Paper, Scissors Game (C++) 🎮

لعبة (حجر - ورقة - مقص) تفاعلية تم تطويرها باستخدام لغة C++ وتعمل عبر الشاشة النصية (Console Application). تم تصميم المشروع كجزء من تطبيقات الكود النظيف والمبادئ البرمجية المنظمة (Clean Code & Modular Programming) بناءً على توجيهات "Programming Advices".

---

## 🚀 الميزات (Features)
* اللعب ضد الكمبيوتر: نظام يقوم باتخاذ خيارات عشوائية ومؤتمتة في كل جولة باستخدام الـ Enums والـ Structs بشكل منظم.
* تتبع الجولات والنتائج: حساب دقيق لعدد جولات المباراة، وعدد مرات فوز اللاعب، وفوز الكمبيوتر، وحالات التعادل.
* تحديد الفائز الإجمالي: إعلان الفائز النهائي بالمباراة بناءً على عدد الجولات المحددة مسبقاً.
* إعادة اللعب: إمكانية تكرار اللعب وخوض مباريات جديدة دون الحاجة لإعادة تشغيل البرنامج يدوياً.

---

## 🛠 التقنيات والمفاهيم المستخدمة
* اللغة الأساسية: C++
* بنية البيانات: Structs & Enums لتمثيل خيارات اللعبة وحالات الفوز.
* أسلوب البرمجة: البرمجة المجزأة (Modular Programming) لتوزيع المهام على دالات (Functions) صغيرة ونظيفة.

---

## 💻 كيف تعمل اللعبة (How to Run)
1. قم بتحميل الملفات أو عمل Clone للمستودع.
2. افتح ملف الحل The Project.sln أو مجلد المشروع عبر بيئة التطوير الخاصة بك.
3. قم بعمل بناء وتشغيل (Compile & Run) للمشروع.
4. اتبع التعليمات المكتوبة على الشاشة لإدخال عدد الجولات واختيار حركتك.
